# windows_local_group_member (Data Source)

Reads the membership state of a specific (group, member) pair by name without
managing the membership lifecycle.

Both `group_name` and `member_name` are required. The resolved SIDs and
principal source are returned as computed attributes.

Returns an error when the group or member is not found on the target host.

## Example Usage

```terraform
# Look up a specific member of the Administrators group by name.
data "windows_local_group_member" "admin_user" {
  group_name  = "Administrators"
  member_name = "Administrator"
}

output "admin_user_sid" {
  value = data.windows_local_group_member.admin_user.member_sid
}

output "admin_user_principal_source" {
  value = data.windows_local_group_member.admin_user.member_principal_source
}
```
