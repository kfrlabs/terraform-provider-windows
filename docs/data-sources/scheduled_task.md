# windows_scheduled_task (Data Source)

Reads the current state of a Windows Scheduled Task by name and path.
Does **not** manage the task lifecycle (no create, update, or delete).

Write-only attributes (`password`, `password_wo_version`) are intentionally
absent from this data source.

Returns a Terraform error when the task is not found on the target host.

## Example Usage

```terraform
# Read a built-in Windows task.
data "windows_scheduled_task" "defrag" {
  name = "ScheduledDefrag"
  path = "\\Microsoft\\Windows\\Defrag\\"
}

output "defrag_enabled" {
  value = data.windows_scheduled_task.defrag.enabled
}

output "defrag_state" {
  value = data.windows_scheduled_task.defrag.state
}

output "defrag_last_run" {
  value = data.windows_scheduled_task.defrag.last_run_time
}

# Inspect a task managed by the windows_scheduled_task resource.
data "windows_scheduled_task" "backup_info" {
  name = windows_scheduled_task.backup.name
  path = windows_scheduled_task.backup.path
}
```
